## :dart: Para Utilizar
Você pode fazer o git clone para clonar o repo em sua máquina.</br></br>
Lembre-se de baixar/atualizar as dependências do projeto com:
```bash
npm
```
ou
```bash
yarn
```

### Storybook
Para rodar o projeto no Storybook.
```bash
npm run storybook
```

### Web
Para rodar o projeto na web.
```bash
npm run dev
```
